# Airbnb Mallorca — Visualización del mercado turístico

Proyecto de visualización de datos centrado en la evolución del mercado Airbnb en Mallorca, explorando su crecimiento, profesionalización y distribución territorial mediante distintas técnicas de visualización interactiva.

Página web del proyecto:  
https://damiaescalas.github.io/mallorca-airbnb-visualization/

La web combina visualizaciones desarrolladas con:
- Tableau Public
- Flourish

---

# Objetivos del proyecto

El análisis busca responder a distintas preguntas relacionadas con la evolución del alquiler turístico en Mallorca:

- ¿Cómo ha evolucionado Airbnb en la isla?
- ¿Qué zonas concentran una mayor actividad?
- ¿Existe una fuerte estacionalidad turística?
- ¿Qué diferencias existen entre tipos de anfitrión?
- ¿Cómo se relacionan precio y valoraciones?

---

# Visualizaciones incluidas

El proyecto incluye distintas visualizaciones interactivas:

- Mapa temporal de expansión de Airbnb
- Evolución acumulada del mercado
- Distribución de anfitriones según perfil
- Estacionalidad de la ocupación turística
- Comparativa territorial entre municipios
- Relación entre valoraciones y precio
- Comparación de puntuaciones según tipo de anfitrión
- Sankey de segmentación del mercado

---

# Clasificación de anfitriones

Para el análisis se definieron tres perfiles de anfitrión:

| Tipo | Descripción |
|---|---|
| Local | Alojamientos concentrados principalmente en Mallorca |
| Mixto | Operan en distintos mercados pero mantienen una presencia importante en Mallorca |
| Global | Mallorca representa solo una pequeña parte de su cartera total |
